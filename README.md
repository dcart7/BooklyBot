# 💅 BooklyBot — Твій ідеальний Telegram-помічник для запису клієнтів

Привіт! Це **BooklyBot** — зручний та багатофункціональний Telegram-бот, створений спеціально для майстрів манікюру, косметологів та інших спеціалістів beauty-сфери. Він бере на себе всю рутину: записує клієнтів, нагадує їм про візит та веде розклад, щоб ти могла зосередитися на головному — красі! ✨

## 🌟 Що вміє цей бот?

### 👩‍🦰 Для твоїх клієнтів:
* **Зручний запис (24/7):** Клієнти самі обирають послугу, дивляться вільні дати у гарному календарику та бронюють зручний час.
* **Мої записи:** Будь-якої миті можна переглянути свій активний запис або скасувати його (тоді віконце одразу стане вільним для інших).
* **Актуальні ціни та портфоліо:** Завжди доступний прайс-лист та посилання на твої роботи.
* **Турботливі нагадування:** Бот сам нагадає клієнту про візит за 24 години. Жодних "ой, я забула"! 🔔

### 👑 Для тебе (Адмін-панель):
Команда `/admin` відкриває твій пульт керування усім ботом:
* **Гнучкий графік:** Легко додавай робочі дні, відкривай чи закривай їх одним натисканням.
* **Керування віконцями (слотами):** Налаштовуй час для запису так, як зручно саме тобі.
* **Усі записи під рукою:** Окремий розділ, де видно всіх, хто записався. Якщо плани змінилися — запис можна легко скасувати, і клієнт отримає повідомлення про це.
* **Редагування інфо:** Прямо з бота можна оновити прайс або змінити посилання на портфоліо в один клік.

> 💡 **Фішка:** Бот стежить, щоб в одного клієнта був лише один активний запис одночасно. А ще, він може перевіряти, чи підписана людина на твій Telegram-канал перед тим, як дозволити запис! Відмінний спосіб збирати аудиторію 😎

## 🚀 Як встановити та запустити (Інструкція для розробників)

Якщо ти хочеш розгорнути цього бота на своєму сервері або комп'ютері, ось покроковий гайд:

### 1. Завантаж проєкт
Відкрий термінал та перейди в папку, де збережено файли:
```bash
cd BooklyBot
```

### 2. Створи віртуальне середовище
Це потрібно, щоб усі Python-бібліотеки встановилися ізольовано і нічого не зламалося:
```bash
python3 -m venv venv
source venv/bin/activate        # Для macOS та Linux
# або: venv\Scripts\activate    # Для Windows
```

### 3. Встанови залежності
```bash
pip install -r requirements.txt
```

### 4. Налаштуй змінні оточення (Конфіг)
Скопіюй шаблон файлу конфігурації, щоб створити свій `.env` файл:
```bash
cp .env.example .env
```
Тепер відкрий `.env` файл та заповни його своїми даними:

| Змінна | Що це таке? |
|---|---|
| `BOT_TOKEN` | Токен твого бота (отримай його у [@BotFather](https://t.me/BotFather)) |
| `ADMIN_ID` | Твій особистий Telegram ID (можна дізнатися через [@userinfobot](https://t.me/userinfobot)) |
| `SCHEDULE_CHANNEL_ID` | ID або `@username` каналу, куди бот скидатиме розклад та нові записи |
| `CHANNEL_ID` | ID або `@username` твого основного каналу (для перевірки обов'язкової підписки) |
| `CHANNEL_LINK` | Посилання-запрошення на канал (`https://t.me/...`) |

⚠️ **Дуже важливо:** Не забудь додати свого бота в адміністратори обох каналів, інакше він не зможе перевіряти підписку та публікувати розклад!

### 5. Поїхали! Запускаємо бота 🚀
```bash
python bot.py
```

## 🛠 З чого він зібраний (Технології)
* **aiogram 3.x** — сучасний та дуже швидкий фреймворк для Telegram-ботів на Python.
* **aiosqlite** — легка вбудована база даних (не треба ставити ніяких важких SQL-серверів).
* **APScheduler** — магія, яка відповідає за відкладені нагадування.
* **python-dotenv** — для безпечного зберігання токенів у `.env` файлі.

Якщо знайдеш баги або матимеш ідеї щодо покращення — feel free to contribute! ❤️
